This page allows you to add a team along with it's mascot and team colors to the team list! Go ahead and test it out and then check out the changes on the team list tab!

It sends every addition to the team list to the backend server. When deleted, it is removed from the server.

You can also send a request to the API on the server to get a list of teams in JSON format.
